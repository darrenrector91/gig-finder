## Gig Finder

<h2>Requirements to start using application</h2>
<ol>
    <li>Google account or an email is necessary to login into application</li>
</ol>

<h2>Steps to launch app</h2>
<ol>
    <li>Using terminal type <code>npm install --save</code> for required dependencies</li>
    <li>While in terminal also <code>npm install nodemon --save-dev</code> </li>
    <li>Start database server</li>
    <li>Spin up Postico</li>
    <li>Create new database</li>
    <li>Create table using <code>jobs.sql</code> file</li>
    <li>Open IDE</li>
    <li>Type <code>npm start</code> or <code>nodemon</code> in terminal to run on browser</li>
</ol>
